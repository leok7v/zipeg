Zipeg

Zipeg program itself is not password protected the file is.

Password is assigned to the archive file by the person who created that file.

If you do not know the password - the only way to learn it is to ask that person.

Passwords are case sensetive. Check that CAPS LOCK!

We cannot help you to recover (a.k.a. "crack") the forgotten password.

Modern RAR and ZIP files use Advanced Encryption Standard (AES) that is cryptographycally strong and virtually impossible to crack.
