Crash report contains technical information that helps developers to identify and fix problem you have experienced.
Crash report does not contain any private information about you.

If you experienced a problem it is always a good idea to check that you have latest version of:

  • Zipeg

  • Java for Mac OS X 10.6 Update 2 (Snow Leopard)
  • Java for Mac OS X 10.5 Update 7 (Leopard)
  • Java for Mac OS X 10.4 (Tiger)
  • Java for Windows (all versions)

Keeping your Java installation up to date also improves security of your Internet browsing.

If you already have the current stable version and your java installation is up to date, you may want to check the latest beta version of Zipeg at http://zipeg.googlecode.com but keep in mind that this is beta build, so mileage can vary.
