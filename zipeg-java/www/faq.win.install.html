Installation on Windows is pretty straight forward and fully automated.

Download ZipegForWindows.exe

Click on it and follow the installer steps.

Do NOT run ZipegForWindows.exe As Administrator.

To uninstall:

  • on Windows 2000/XP use Control Panel / Add Remove Programs / Zipeg.
  • on Windows Vista use Control Panel / Programs And Features / Select Zipeg / Uninstall.
